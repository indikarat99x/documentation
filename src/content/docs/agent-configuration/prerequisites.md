title: Prerequisites
description: Control plane access, a running agent, and repo admin rights — everything to have ready before you begin.
---

Before you start the [Quick Start](../quickstart) guide, make sure the following are in place. Having these ready turns installation into a smooth, five-minute process.

| What you need | Why |
|---|---|
| **Xians control plane** — run your own via [xians.ai](https://xians.ai), or use a hosted option like 99x's [Agentri](https://agentri.ai/) | Provides the runtime environment that orchestrates your agents |
| **Xianix Agent** — a deployed instance on your Xians control plane; on Agentri, one is already available for 99x teams | The agent that will execute tasks such as PR review on your repositories |
| **Git repository access** — admin-level permissions on GitHub and/or Azure DevOps | Required to configure webhooks and grant the agent access to your repositories |

:::note
If you're a 99x team member using Agentri, the control plane and agent are already provisioned for you — you only need Git repository admin access to get started.
:::
